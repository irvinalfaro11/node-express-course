### Task Manager - Web Apps

- ⚡ Node Js - Express Js
- ⚡ MongoDB

### How to run

- Set MONGO_URI in NODE ENV, containing MongoDB connection string.
- Set PORT via NODE ENV to avoid port conflicts (default: 3000)
